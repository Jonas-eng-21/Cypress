# Cypress
Estudos com o framework Cypress
***
Passos para a execução:

*Abrir em terminais diferentes e executar os códigos:*
* Terminal 1:
  * cd backend
  * npm start
* Terminal 2: 
  * cd frontend
  * npm start
* Terminal 3: 
  * npm run cy:open


# PageObjects
